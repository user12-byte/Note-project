# Note-project
 
## 项目背景
 
针对个人信息散乱、难以统一管理的需求，设计并实现一个基于网页的便签系统，支持在本地浏览器中进行信息的录入、查询、分类与管理。
 
技术选型：**Flask（后端）+ SQLite（数据库）+ 原生 HTML/CSS/JS（前端）**
 
## 实现的功能
 
- **信息录入**：创建便签，支持标题、正文、分类标签
- **图片上传**：便签中可附加图片
- **分类管理**：按类别筛选查看便签
- **查询**：关键词搜索便签内容
- **删除**：移除不需要的便签
- **本地部署**：启动后通过浏览器访问，支持本地用户使用
 

这是一个练手项目，主要验证 Flask 做 Web 应用的基本流程，代码结构比较简单，本来想连MySQL没做，想打包失败了。
